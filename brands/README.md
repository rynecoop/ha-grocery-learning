# Brand Assets for HACS/Home Assistant icon

This directory contains in-repo brand assets used by HACS validation.
For default HACS inclusion, also submit these assets to the Home Assistant brands repository.

Required target in brands repo:
- `core_integrations/grocery_learning/`

Recommended files:
- `icon.png` (256x256)
- `logo.png` (256x256)

After the brands PR is merged and Home Assistant updates brand assets, the integration icon will show automatically.
