# Branding for HACS/Home Assistant icon

To show a custom icon for this integration in Home Assistant/HACS, submit branding assets to the Home Assistant brands repository.

Required target in brands repo:
- `core_integrations/grocery_learning/`

Recommended files:
- `icon.png` (256x256)
- `logo.png` (256x256)
- Optional source: `icon.svg`

This folder includes a starter SVG:
- `local-grocery-assistant-icon.svg`

After the brands PR is merged and Home Assistant updates brand assets, the integration icon will show automatically.
