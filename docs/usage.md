# Usage Guide

## Daily Flow
1. Add items with Quick Add.
2. Add items by voice (shopping list/grocery list aliases are auto-routed).
3. Shop by category sections.
4. Check off items as you pick them.
5. Clear completed items when done.

## Duplicate Handling
- If an item already exists, you get a duplicate decision card.
- Choose `Add Anyway` or `Skip`.

## Review and Learning
- Unknown items may route to `Other`.
- Use review actions to move and learn once.
- Future adds of the same term should auto-route correctly.

## Recategorize an Item
1. Click an item row to open category controls.
2. Select target category.
3. Click `Move`.
4. Demo: [change-category.mp4](videos/change-category.mp4)

## Completed Items
- Completed items move into the completed section.
- Uncheck from completed to restore.
- Use `Clear Completed` to purge history quickly.
- Demo: [completed-clear.mp4](videos/completed-clear.mp4)
