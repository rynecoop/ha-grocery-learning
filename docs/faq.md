# FAQ

## Is this local-only?
Yes. The Home Assistant version is designed to run inside Home Assistant itself.

## Do I need YAML helpers?
No. The integration provisions and manages what it needs.

## Can I reorder categories later?
Yes. Open `List Settings`, reorder them, and save.

## Can I rename a category later?
Yes. Category names can be edited in place from `List Settings`.

## Does renaming a category move existing items into the new name?
Yes. Existing items keep the renamed category instead of falling back to `other`.

## Can multiple devices stay in sync while the app is open?
Yes. The panel now refreshes near-real-time across open Home Assistant devices.

## Will this work with voice assistants?
Yes, when voice adds create `todo.add_item` calls in Home Assistant.

## Does it track who added items?
Quick Add and routed adds include source/user attribution where Home Assistant provides that context.

## Can I clear completed history?
Yes. Use `Clear Completed` in the app.
