# FAQ

## Do I need YAML helpers?
No. This integration provisions and manages what it needs.

## Can I reorder categories later?
Yes. Open `Configure`, edit category list order, save.

## Is this local-first?
Yes. Routing and app behavior run in Home Assistant.

## Will this work with voice assistants?
Yes, when voice adds create `todo.add_item` calls in Home Assistant.

## Does it track who added items?
Quick Add and routed adds include source/user attribution where available.

## Can I clear completed history?
Yes. Use `Clear Completed` in the app.
