## Capture Rules
1. use the current Home Assistant release with the current HACS version of Local List Assist
2. show current labels:
   - `App Settings`
   - `List Settings`
   - `Activity`
3. avoid old `Configure` wording
4. avoid outdated category edit controls
5. if the dashboard name is customized, make sure it is intentional
6. keep personal data and unrelated entities out of frame where possible

## Minimum Acceptance
- screenshots match the current UI shape and navigation
- videos show the current behavior without cuts that hide the important state change
- live refresh demo clearly shows two open HA clients
